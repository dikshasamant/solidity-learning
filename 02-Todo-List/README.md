# Todo List Smart Contract

## Description
A Solidity smart contract for managing tasks.

## Features
- Add a task
- Get a task by ID
- Get all tasks
- Count tasks
- Mark a task as completed
- Update task title
- Delete a task (Swap & Pop)

## Concepts Practiced
- Structs
- Arrays
- Mappings
- CRUD Operations
- require()
- Loops
- delete
- push()
- pop()
- Swap & Pop deletion

## Solidity Version
0.8.18
