# Student Management Smart Contract

## Description

A simple Solidity smart contract for managing student records.

## Features

- Add new students
- Retrieve student by roll number
- Retrieve all students
- Update student marks
- Count total students
- Input validation using `require()`

## Solidity Concepts Used

- Structs
- Arrays
- Mappings
- Functions
- Memory
- State Variables
- For Loops
- Require Statements
